# 🏎️ NFS Portal 🏁


**NFS PORTAL** — это высокотехнологичный портал для гонщиков Рокпорта, объединяющий в себе функции мессенджера, социальной сети и системы отслеживания игроков в реальном времени. Дизайн полностью вдохновлен интерфейсом игры *Need for Speed: Most Wanted (2005)*.

---

## 🚀 Основной функционал

### 📡 Система "Маяк" (Real-time Presence)
Сердце проекта — глобальный радар. Благодаря интеграции с **Supabase Realtime**, пользователи видят статусы друг друга без перезагрузки страницы:
- 🟢 **ONLINE**: Пользователь на сайте.
- 🔴 **В ЧАТЕ**: Находится в мессенджере (отслеживается по `location`).
- 🟡 **В ИГРЕ**: Ищет соперников или уже находится в заезде.

### 💬 Messenger & Social
- **Приватные и общие чаты**: Полноценная система обмена сообщениями.
- **Индикатор печати**: Вы видите, когда собеседник набирает текст.
- **Умные бейджи**: Индикация непрочитанных сообщений, которая сбрасывается только при прочтении.
- **Друзья**: Система запросов в друзья с мгновенными уведомлениями.

### 🛠 Админ-панель (Blacklist Control)
Интегрированная система модерации:
- Возможность удаления сообщений (для администраторов).
- Визуальное выделение (Glow effect) для спец-ролей.

## 💻 Технический стек

| Технология | Назначение |
| :--- | :--- |
| **JavaScript (ES6+)** | Логика фронтенда и работа с API |
| **Supabase** | База данных (PostgreSQL), Auth и Realtime-каналы |
| **CSS3** | Кастомная стилизация, анимации в стиле NFS, адаптивная верстка |
| **SweetAlert2** | Стилизованные игровые уведомления (Toasts) |
